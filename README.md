# Windows 8.1 to Windows 8 Release Preview Converter
<sup>**Windows 8.1 to Windows 8 RP Converter**/**Windows 8.1 to 8 RP Converter**</sup>

<sup>made with Visual Basic .NET in Visual Studio 2012 in Windows 8.1</sub>

This program converts Windows 8.1 to Windows 8 Release Preview (also known as Windows 8 Release Candidate or Windows 8 Build 8400) by applying a theme to Windows 8.1, as well as installing additional programs and replacing some system files with ones of Windows 8 RP and RTM, to make Windows 8.1 look and behave almost like this pre-release version of Windows 8.

For more information and pictures of this version on the BetaWiki, [click here](https://betawiki.net/wiki/Windows_8_build_8400)
